# astrbot_plugin_porntrex

AstrBot Porntrex API 插件

## 命令

- `/pt_video <video_url>` - 获取视频信息（传入完整 URL）
- `/pt_model <model_id>` - 获取模特信息
- `/pt_channel <channel_id>` - 获取频道信息
- `/pt_search <query>` - 搜索视频（返回完整 URL）

## 配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| proxy | 代理地址 | 空 |
| blur_level | 图片打码程度 (0-100) | 20 |

## 安装

在 AstrBot 管理面板中添加插件仓库地址：

```
https://github.com/vmoranv/astrbot_plugin_porntrex
```
